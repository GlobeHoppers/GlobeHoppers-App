# GlobeHoppers v7.5.15 Release Notes

- Converted Screensaver Mode into a presentation-only interface.
- Hidden the top navigation, Add Hop, Hoppers, GlobeHopper Timeline, Globe, Fullscreen, and Play controls while Screensaver Mode is active.
- Hidden About text, map zoom diagnostics, authoring surfaces, search, advanced controls, and Globe View controls in Screensaver Mode.
- Preserved the current and queued Hop cards as playback context.
- Extended the timeline across the complete bottom edge of the viewport.
- Removed the screensaver timeline's side gaps and rounded outer corners while retaining its pins, dates, progress, and active-Hop callout.
- Preserved the v7.5.14 automatic timeline-to-Globe-View cycle and all v7.5.13 playback behavior.
