# GlobeHoppers v7.5.12 Release Summary

GlobeHoppers v7.5.12 is a focused information and presentation update built on the stable v7.5.11 camera and route baseline.

- Added compact Plane, Car, Train, and Boat indicators to Timeline search and Choose-a-Hop cards.
- Extended the active timeline pin stem by 100 percent while keeping the callout attached to the raised pin head.
- Scoped Routes only and Locations only filters to explicit Globe View so playback always restores the complete route-and-location scene.
