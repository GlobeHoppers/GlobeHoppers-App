# GlobeHoppers v8.2 Work Package 4

## Scope

Enables transactional editing of an individual private trip. The update replaces trip metadata, ordered legs, route geometry, and Hopper assignments in one database transaction.

## Security

- Uses `auth.uid()` through `private.can_edit_map`.
- Requires the trip to belong to the selected map.
- Validates every Hopper belongs to the map.
- Uses `updated_at` optimistic concurrency protection.
- Cross-account edits are rejected.
