# GlobeHoppers v7.5.10 Release Notes

GlobeHoppers v7.5.10 is a comprehensive camera and timeline stabilization release based on v7.5.9.

## Camera

- Distinguishes genuine user camera gestures from MapLibre events generated by playback `jumpTo` and `easeTo` calls.
- Allows exactly one camera-reacquisition request per completed user gesture.
- Replaces staged fixed-duration return motion with a live-target exponential-damping controller.
- Keeps the camera at a safe outside-globe zoom until its orientation is substantially restored.
- Preserves the normal vessel render and playback-follow loop throughout reacquisition.
- Prevents initial playback camera movement from being mistaken for a manual pan.

## Timeline

- Replaces the inherited timeline marker layout with an isolated fixed-geometry timeline implementation.
- Timeline zoom changes horizontal spacing only. Pins, years, months, and rail height do not move or resize vertically.
- Anchors the active pin and location pill to the actual rendered pin position.
- Removes the duplicate active-pin visual from the base rail while the overlay is visible.
- Keeps years and trip-associated month labels inside fixed, unclipped rows.
- Keeps first and last pins fully visible at Fit.
- Reduces timeline padding and overall control-bar thickness.
