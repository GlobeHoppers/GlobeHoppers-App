# GlobeHoppers v8.2.4 — Work Package 4 Timeline Access Hotfix

This hotfix restores access to the GlobeHopper Timeline in private cloud mode so users can select an individual trip and open Edit Hop.

Cloud-mode safeguards remain in place:

- Individual Add Hop is enabled.
- Individual Edit Hop is enabled.
- Delete Hop remains hidden and disabled.
- Batch Add remains disabled.
- Timeline reordering remains disabled.
- Legacy GitHub repository controls remain hidden.
