# GlobeHoppers v3.0 — GlobeHoppers Studio UI Overhaul

## Highlights

- Renames the old Admin Mode interface to **GlobeHoppers Studio**.
- Adds a polished right-side Edit Travel History panel.
- Adds centered Add Trip and Edit Trip modals.
- Save commits updated travel data to GitHub, then closes the modal.
- Adds traveler toggle buttons for Joey and Bonnie. Selecting both creates a Joey + Bonnie trip.
- Adds icon-style selectors for Plane, Car, Train, and Boat.
- Adds destination text entry with autocomplete from known locations.
- Allows new destination text entries and stores them as locations marked `needsGeocoding` for future Mapbox Geocoding support.
- Adds optional From override plus Additional Legs for chained trips. Chained routes override the round-trip toggle.
- Adds Reorder mode with drag-and-drop within date buckets.
- Unknown month trips continue to sort at the end of their year.
- Adds an upload photos placeholder for the future media workflow.
- Moves repository/token/commit/download tools into a collapsed Repository Settings section.
- Darkens dropdown/select menus to match the GlobeHoppers theme.

Repository path and GitHub Pages deployment remain unchanged.
