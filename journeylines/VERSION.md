- Edit GlobeHopper Timeline no longer pauses playback just by opening the drawer
- Playback only pauses when Add Hop or Edit Hop modal opens
- Edit GlobeHopper Timeline now receives the active trip and highlights the current row
- Mixed non-squad trips show secondary Hopper/Guest color accents in the edit timeline row corner
- Active trip card now colors each mixed Hopper/Guest name with that Hopper/Guest color
- Timeline, Edit Timeline, and Edit Hoppers menus are slightly more transparent/glass-like
- Date cells wrap more aggressively so longer dates like September 2025 / December 2025 do not collide
- Guest Hopper color popover is nudged inward to avoid clipping
- package intentionally omits src/data/trips.json, src/data/hoppers.json, and package-lock.json
