# JourneyLines Routing Notes — v2.15

Driving routes use Mapbox Directions when a public Mapbox token is available. The deployed app reads the token from `/JourneyLines/runtime-config.js`, which is generated by GitHub Actions from the repository secret named `VITE_MAPBOX_TOKEN`.

After deploying, confirm:

```text
https://jonathanjoelneptune.github.io/JourneyLines/runtime-config.js
```

It should contain a `pk.` token and `tokenSource: github-actions-secret-dist-runtime-config`.

If it still shows an empty token, check the latest GitHub Actions log. In v2.15, the workflow should fail before publishing if `VITE_MAPBOX_TOKEN` is unavailable. If it does not fail, the updated root `.github/workflows/deploy.yml` was not uploaded.

Mapbox driving geometry is cached in browser localStorage under the route cache version `v2.15`.
